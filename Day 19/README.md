# OpenROAD Installation

**Cloning below the git repository**
```
git clone --recursive https://github.com/The-OpenROAD-Project/OpenROAD-flow-scripts
```
![img1]()  

**Running setup script**
```
cd OpenROAD-flow-scripts
sudo ./setup.sh
```
![img2]()   

**Build OpenRoad**  
```
./build_openroad.sh --local
```
